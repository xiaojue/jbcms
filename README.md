# jbcms

基于nodejs的cms系统

## 介绍

jbcms取名来自于js build cms的简写，是一款采用nodejs来开发的强大的内容管理系统，力求简单，方便，轻巧，易扩展。
本项目采用WTFPL公共许可证，我们鼓励你们"想他妈的干嘛就干嘛"的一种程序员精神。
